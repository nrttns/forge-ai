## 1. Purpose

This document defines the active phase and stage sequence for development of the Forge AI repository as a self-hosted Target Repository.

It is Forge AI repository-specific project planning. It does not define universal physical paths or a mandatory phase structure for external Target Repositories. External Target Repositories declare their own DevelopmentPhases resources through their own root `AGENTS.md` and project truth.
