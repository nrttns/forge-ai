# AI-DOS Product Navigation

This document is the active navigation entry for the reusable AI-DOS product under `docs/AI/`. It routes consumers to governance, architecture, System Layer, Operational Core, Commands, Workflows, and Templates without creating authority.

## Product and Target Boundary

- `docs/AI/` owns reusable AI-DOS product truth.
- The active Target Project owns its source, ProjectStatus, DevelopmentPhases, Roadmap, protected areas, evidence, secrets, and live state.
- Forge AI is the self-application Product Development Target Project; it is not AI-DOS itself.
