# Forge AI Governance Records

This directory is reserved for Forge AI Target Project governance records and governance evidence.

Human Governance remains the final authority for Forge AI approvals, scope changes, protected-area exceptions, and autonomy claims. Target Project governance evidence records decisions, approvals, review outcomes, blockers, and acceptance evidence that support Human Governance decisions.

Future records may include decision logs, approval records, review dispositions, protected-area exception records, and autonomy acceptance evidence.

AI-DOS governance truth remains under the appropriate `docs/AI-DOS/` authority. This directory must not redefine AI-DOS governance.

This migration creates no new governance policy.
