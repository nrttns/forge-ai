# Forge AI Reports

This directory is reserved for Forge AI Target Project reports and evidence.

Forge AI Target Project operational reports must live outside AI-DOS product truth when they record Forge AI project execution, validation, review, blockers, acceptance, or Target Project state.

AI-DOS product-level architecture, purity, standards, Runtime, System, and product compliance reports remain under the appropriate `docs/AI-DOS/` authority.

Target Project operational reports must not be stored as AI-DOS product truth.

Existing reports are not automatically moved by this migration. Future disposition requires separate evidence and Human Governance authorization.
