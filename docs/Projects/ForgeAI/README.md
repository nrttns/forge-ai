# Forge AI Target Project Documentation Index

This directory contains Forge AI Target Project truth. It is the canonical documentation boundary for Forge AI mission, planning, governance evidence, reports, and Target Project navigation.

It is not an AI-DOS product documentation index, a second `AGENTS.md`, a planning authority, a live ProjectStatus, or an AI-DOS invocation implementation.

## Directory Map

| Area | Path | Purpose |
|:---|:---|:---|
| Mission | `docs/Projects/ForgeAI/Mission/` | Mission authority and the canonical Forge AI Target Project Contract. |
| Planning | `docs/Projects/ForgeAI/Planning/` | DevelopmentPhases, Roadmap, and ProjectStatus for Forge AI Target Project planning and state. |
| Governance | `docs/Projects/ForgeAI/Governance/` | Future Forge AI Target Project governance records and governance evidence. |
| Reports | `docs/Projects/ForgeAI/Reports/` | Future Forge AI Target Project reports and operational evidence. |
